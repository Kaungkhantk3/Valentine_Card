# Unlock Animation Videos

Place your demo videos here:

1. `find-heart-demo.mp4` - Demo video for "Find the Real Heart" unlock animation (~4 seconds)
2. `bring-together-demo.mp4` - Demo video for "Bring Together" unlock animation (~4 seconds)

## Video Specifications

- Duration: ~4 seconds
- Format: MP4
- Aspect Ratio: 9:16 (portrait/vertical)
- Resolution: 720x1280 or higher recommended
- The videos will loop automatically in the preview

## Note

If the video files are not present, a fallback icon will be displayed instead.
